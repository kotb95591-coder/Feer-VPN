Картинки для бота Feer VPN
========================

Сохрани сюда 3 файла с точными именами:

  welcome.png   — логотип FEER VPN (приветствие /start и главное меню)
  tariffs.png   — баннер «ВЫБЕРИ ТАРИФ» (экран тарифов)
  howto.png     — баннер «V2/H — Как подключится?» (экран «Как подключиться»)

Формат: PNG или JPG. Рекомендуемый размер — до 1280 px по длинной стороне, до 5 МБ.

Пути к файлам настраиваются в .env (IMG_WELCOME / IMG_TARIFFS / IMG_HOWTO).
Если файл не найден — бот просто покажет текст без картинки (ничего не сломается).

ВАЖНО для Vercel: эта папка должна попасть в git (она НЕ в .gitignore),
иначе на хостинге картинок не будет. Либо загрузи картинки куда-нибудь
и укажи прямые URL в IMG_* переменных.

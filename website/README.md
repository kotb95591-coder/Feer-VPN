# FeerVPN — лендинг

Премиальный одностраничный сайт FeerVPN. Чёрно-белый стиль, адаптивная вёрстка, без сборки.

## Состав

- `index.html` — весь сайт в одном файле (HTML + CSS внутри). Шрифты подгружаются с Google Fonts.
- `vercel.json` — конфиг для статического хостинга.

## Локальный просмотр

Просто откройте `index.html` в браузере. Либо поднимите локальный сервер:

```bash
python -m http.server 5500
# затем откройте http://localhost:5500
```

## Деплой на Vercel (отдельный репозиторий)

1. Создайте новый репозиторий и скопируйте туда содержимое этой папки.
2. В Vercel → Add New → Project → импортируйте репозиторий.
3. Framework Preset: **Other** (статика). Build Command оставьте пустым, Output Directory — корень (`.`).
4. Deploy. Готово.

Позже можно привязать свой домен в настройках проекта Vercel.

## Что легко поменять

- Ссылка на бота: поиск `https://t.me/feervpn_bot` в `index.html`.
- Цены: блок `#pricing` (Solo 100₽, Семья 249₽).
- Цвета и радиусы: CSS-переменные в `:root` в начале `index.html`.
